# JARVIS

**JARVIS** — это голосовой ассистент для Windows с футуристичным HUD-интерфейсом, быстрыми локальными командами, умным режимом общения, поиском информации, изображений и удобным управлением компьютером голосом.

Программа создана для тех, кто хочет сделать работу за ПК быстрее, удобнее и эффектнее: запускать приложения, управлять системой, искать информацию, получать ответы на вопросы и взаимодействовать с компьютером почти как с настоящим ассистентом.

---

## Возможности

### Голосовое управление
JARVIS понимает голосовые команды и помогает управлять компьютером без постоянного использования мыши и клавиатуры.

Примеры:
- открыть блокнот, калькулятор, браузер, Steam, Discord и другие приложения
- открыть сайты и сервисы
- сделать скриншот
- изменить громкость
- работать с окнами и функциями Windows

### Два режима работы

#### FAST
Быстрый режим для повседневного использования.

Подходит для:
- локальных команд
- быстрого запуска приложений
- системных действий
- минимальной нагрузки на ПК

#### JARVIS
Полный режим с расширенными возможностями.

Включает:
- умные ответы
- поиск информации
- поиск изображений
- визуальные карточки результатов
- более “живое” и эффектное общение

### Умный поиск и ответы
В режиме **JARVIS** ассистент умеет:
- отвечать на вопросы
- кратко объяснять понятия
- искать информацию
- показывать изображения
- помогать открывать найденные результаты

Примеры:
- `что такое DLSS`
- `найди фото Марса`
- `сравни Steam и Epic Games`
- `покажи картинки Ferrari`

### Красивый HUD-интерфейс
JARVIS использует анимированный интерфейс в стиле AI/HUD.

В интерфейсе отображаются:
- состояние ассистента
- уведомления
- результаты поиска
- изображения
- статус обновлений
- системные ответы

### Более естественное общение
JARVIS умеет:
- реагировать во время собственной речи
- воспринимать команды остановки
- прерываться и слушать пользователя
- делать общение более плавным и похожим на настоящего ассистента

### Автообновление
JARVIS поддерживает обновление через установщик и GitHub Releases.

Пользователь может:
- проверить наличие новой версии
- получить уведомление об обновлении
- обновить программу через установщик

## Установка

### Вариант 1. Через установщик
1. Скачайте последнюю версию установщика из раздела **Releases**
2. Запустите `JARVIS_Installer.exe`
3. Установите программу в удобную папку
4. Запустите JARVIS

### Вариант 2. Для разработки
1. Клонируйте репозиторий
2. Установите Python
3. Создайте виртуальное окружение
4. Установите зависимости из `requirements.txt`
5. Запустите проект

Пример:

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
