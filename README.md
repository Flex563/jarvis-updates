# JARVIS Updates

Публичный репозиторий обновлений JARVIS.

Здесь публикуются только релизы и установщик программы.
Исходный код основного проекта в этот репозиторий не входит.

## Как обновляется JARVIS

JARVIS и установщик проверяют последнюю доступную версию через GitHub Releases.

Последний установщик:
`https://github.com/ТВОЙ_ЛОГИН/jarvis-updates/releases/latest/download/JARVIS_Installer.exe`

Страница последнего релиза:
`https://github.com/ТВОЙ_ЛОГИН/jarvis-updates/releases/latest`

## Что находится в релизах

В каждый релиз загружается файл:

- `JARVIS_Installer.exe`

Рекомендуется сохранять одинаковое имя установщика в каждом новом релизе.

## Формат версий

Используй теги в формате:

- `v1.0.0`
- `v1.0.1`
- `v1.1.0`

## Как выпустить новую версию

1. Собрать новый установщик `JARVIS_Installer.exe`
2. Открыть вкладку **Releases**
3. Нажать **Draft a new release**
4. Создать новый тег, например `v1.0.1`
5. Прикрепить `JARVIS_Installer.exe`
6. Добавить список изменений
7. Опубликовать релиз

## Пример списка изменений

### v1.0.1
- исправлен запуск системных приложений Windows
- улучшен поиск установленных программ
- добавлена проверка обновлений
- повышена стабильность установщика

## Важно

Этот репозиторий предназначен только для распространения обновлений.
Не используйте его для хранения приватных данных, локальных настроек, кэша и рабочих файлов разработки.
